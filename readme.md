# Ros2 Dev Environment

The aim of this repo is to provide a template for ros projects using a docker container.
Whenever creating a new repo for a project, use this as a template.

## Quickstart
Just install vscode dev containers plugin and follow the last steps [here](
https://docs.ros.org/en/humble/How-To-Guides/Setup-ROS-2-with-VSCode-and-Docker-Container.html)
